AWS認定 機械学習

AWS Certified Machine Learning – Specialty

AWS上で機械学習ソリューションを設計・実装・運用する能力を認定する専門資格。SageMakerをはじめAWSのMLサービスの実務知識を示せ、クラウドML案件で高く評価されます。

試験の基本情報

主催Amazon Web Services（AWS）
レベル上級(Specialty)
試験形式テストセンター/オンライン・65問/180分
受験料300 USD(約45,000円前後)
実施頻度通年
合格率の目安非公開(難度高め)
勉強時間の目安80〜150時間

よくある質問

AWS ML Specialtyは難しいですか？+
上位(Specialty)資格で、機械学習とAWSサービスの双方の知識が問われます。ML経験者でも80〜150時間の学習が目安です。
取得するとどんな案件に有利ですか？+
クラウド上でのML基盤構築・MLOps案件で評価されます。インフラ×MLのスキルを示せるため、市場価値の向上につながります。
