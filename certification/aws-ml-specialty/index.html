AWS認定 機械学習

AWS Certified Machine Learning – Specialty

【2026年3月31日廃止】AWS Certified Machine Learning - Specialtyは廃止されました。後継はML Engineer - AssociateとAI Practitionerです。本ページは記録として残しつつ、後継資格への移行情報を案内します。

試験の基本情報

主催Amazon Web Services（AWS）
レベル上級(Specialty)
試験形式テストセンター/オンライン・65問/180分
受験料300 USD(約45,000円前後)
実施頻度通年
合格率の目安非公開(難度高め)
勉強時間の目安80〜150時間

公式試験データ

2026年6月12日確認 ／ 出典: AWS公式サイト (aws.amazon.com/jp/certification)

  • ✓ 重要: AWS Certified Machine Learning - Specialty は2026年3月31日をもって廃止されました(AWS公式発表)
  • ✓後継・代替はAWS Certified Machine Learning Engineer - Associate および AWS Certified AI Practitioner
  • ✓(参考・廃止前仕様) 受験料300USD・180分・65問
  • ✓取得済みの認定は失効まで有効。これから学ぶ方は後継資格を選んでください

よくある質問

AWS ML Specialtyは難しいですか？+
上位(Specialty)資格で、機械学習とAWSサービスの双方の知識が問われます。ML経験者でも80〜150時間の学習が目安です。
取得するとどんな案件に有利ですか？+
クラウド上でのML基盤構築・MLOps案件で評価されます。インフラ×MLのスキルを示せるため、市場価値の向上につながります。
